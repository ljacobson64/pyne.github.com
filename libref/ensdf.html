ENSDF File Support – pyne.ensdf¶

Pyne should nominally contain support for reading and writing Evaluated Nuclear Structure Data Files (ENSDF). However, currently only parsing out half-life data is supported.

All functionality may be found in the ensdf module:

from pyne import ensdf

Half Lives¶

pyne.ensdf.half_life(ensdf)[source]¶

Grabs the half-lives from an ENSDF file.

Parameters :

ensdf : str or file-like object

ENSDF file to inspect for half-life data
Returns :

data : list of 5-tuples

List of tuples where the indices match

  1. from_nuclide, int (zzaaam)
  2. level, float (MeV) - from_nuc’s energy level
  3. to_nuclide, int (zzaaam)
  4. half_life, float (seconds)
  5. branch_ratio, float (frac)
