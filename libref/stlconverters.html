C++ Standard Library Converters – pyne.stlconverters¶

This module contains wrapper classes for commonly used constructs in the C++ standard library. Becasue Cython does not yet do templating, these classes must be declared and defined for every type.

All functionality may be found in the stlconverters module:

from pyne import stlconverters

This module is largely used by PyNE under the convers, in Cython and elsewhere. However, these classes are of more general interest so feel free to use them in your own code as well.

Set Proxies¶

class pyne.stlconverters.SetInt¶

Wrapper class for C++ standard library sets of type <int>. Provides set like interface on the Python level.

Parameters :

new_set : bool or dict-like

Boolean on whether to make a new set or not, or set-like object with keys and values which are castable to the appropriate type.

free_set : bool

Flag for whether the pointer to the C++ set should be deallocated when the wrapper is dereferenced.
