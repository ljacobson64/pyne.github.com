double molecular_weight(double apm = -1.0)

Calculates the atomic weight of this material based on the composition and the number of atoms per mol. If apm is non-negative then it is used (and stored on the instance) as the atoms_per_mol for this calculation. If apm and atoms_per_mol on this instance are both negative, then the best guess value calculated from the normailized composition is used here.

The ever-important material class and related helpers.

The material class is effectively a normalized nuclide linked list with associated mass, density, atoms per mol, and metadata. However, this implementation also contains other functions for mixing materials and generating related materials.
